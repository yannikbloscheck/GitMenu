# Code of Conduct

This project should provide a welcoming and professional environment for everybody.  
Therefore it's important to always be respectful of differing viewpoints and experiences.

To make it a comfortable, harassment-free experience for everybody you're agreeing to this rules:
* Criticism should always be constructive and never personal
* Trolling, insults and all other attacks are forbidden
* It's unacceptable to use sexual language or imagery and to behave in a sexual way towards others
* No bodily harm should be inflicted on others
* The publishing of any private information without explicit permission is prohibited
* Any other behaviour that's deemed unprofessional isn't allowed

If anybody doesn't adhere to this code of conduct, appropriate corrective action will taken against them.  
Please report any unacceptable behaviour directly to [gitmenu@yannikbloscheck.com](mailto:gitmenu@yannikbloscheck.com). Thank you!
